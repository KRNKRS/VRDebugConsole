# VRDebugLog
これはVR開発上でデバッグの補助を行うアセットです。<br>
現在、実装されている機能は以下。<br>
## 機能
<ul>
<li>リサイズ</li>
<li>ビルボード</li>
<li>最小化</li>
<li>ログのクリア</li>
<li>タイプごとのログの表示/非表示切り替え</li>
<li>項目選択によるスタックトレース表示</li>
<li>Viveコントローラによる限定的な操作</li>
</ul>
